# 📊 Business Intelligence Learning Repository

Selamat datang di repositori **Business Intelligence**!  
Repositori ini berisi kumpulan materi, studi kasus, dan latihan praktis untuk mempelajari Business Intelligence dari dasar hingga penerapan profesional, menggunakan berbagai tools pendukung analisis data.

🔗 **Live Dashboard & Demo:** [faruq1997.github.io/Bisnis_Intelligence/](https://faruq1997.github.io/Bisnis_Intelligence/)

---

## 🧠 Tujuan Repositori

- Memberikan pemahaman konsep dasar Business Intelligence dan Data Mining.
- Menyediakan studi kasus nyata seperti penjualan produk skincare `Fruit Glow` dan data akademik.
- Mengembangkan kemampuan eksplorasi data, visualisasi, dan pengambilan keputusan.
- Menyediakan simulasi interaktif (Quiz & Playground) untuk menguji pemahaman.

## 📌 Topik yang Dipelajari

- 📖 **Literasi Data:** Dasar-dasar BI dan peran Analyst di industri.
- 💽 **Arsitektur Data:** Data Warehouse & Star Schema.
- 📊 **Visualisasi:** Dashboarding menggunakan Excel dan Python (Pandas, Seaborn).
- 🔍 **EDA:** Exploratory Data Analysis untuk mencari insight bisnis.
- 📦 **Strategi Bisnis:** Analisis Produk, Bundling, dan Kanal Distribusi.
- 💡 **DSS:** Decision Support System & Pengambilan Keputusan.
- 📈 **Data Mining:** Klasifikasi, Clustering, dan Market Segmentation.

---

## 🛠 Tools yang Digunakan

- **Excel:** Pivot Table, Power Query, dan Dashboarding Dasar.
- **Python:** Library Pandas, Matplotlib, Seaborn, Plotly, Scikit-Learn.
- **HTML/JS:** Untuk simulasi kuis dan dashboard interaktif ringan.
- **Colab / Jupyter:** Lingkungan eksekusi kode analisis.

## 🧩 Cara Menggunakan

1. **Clone repositori ini:**
   ```bash
   git clone [https://github.com/faruq1997/Bisnis_Intelligence.git](https://github.com/faruq1997/Bisnis_Intelligence.git)
   ```
2. Akses Dashboard: Buka index.html di browser atau kunjungi Link GitHub Pages.
3. Eksekusi Notebook: Buka file .ipynb via Google Colab atau Jupyter Lab lokal.
4. Pelajari Dataset: Gunakan file .xlsx yang tersedia sebagai bahan praktikum mandiri.
